# presence.md

**presence.md** is a simple HTTP server that generates a blog from markdown files.

## Features

* In-memory cache for articles (fast!)
* Hot-reload (for both markdown files and config)
* Archive
* RSS feed
* Tags
* Gzip compression
* TLS support
